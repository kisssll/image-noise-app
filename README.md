# Image Noise Processing Web Application

Веб-приложение для добавления шума к изображениям и анализа распределения цветов.

## Функциональность

- Загрузка изображений
- Добавление шума с заданным уровнем
- Генерация CAPTCHA для защиты от ботов
- Визуализация распределения цветов через гистограммы
- Адаптивный веб-интерфейс

## Технологии

- **Backend**: FastAPI (Python)
- **Frontend**: HTML, CSS, Jinja2 templates
- **Image Processing**: Pillow, NumPy, Matplotlib
- **Testing**: Pytest
- **CI/CD**: GitHub Actions
- **Deployment**: Render.com
